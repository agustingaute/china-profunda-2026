# China Profunda 2026 — Itinerario de Viaje

Landing page estática con el itinerario completo del viaje a China (26 sept – 14 oct 2026).

## Subir a Netlify Drop en 3 pasos

1. **Abrí [app.netlify.com/drop](https://app.netlify.com/drop)** en tu browser (creá cuenta gratis si no tenés).

2. **Arrastrá la carpeta `CHINA`** (la que contiene `index.html`) directo a la zona de drop de Netlify.

3. **Listo.** Netlify te da una URL pública tipo `https://random-name.netlify.app`. Desde el dashboard podés cambiar el nombre del sitio a algo como `china-profunda-2026.netlify.app`.

## Estructura

```
CHINA/
├── index.html   ← único archivo, todo incluido
└── README.md
```

No requiere build, bundler, ni servidor. Todo el CSS, JS y el mapa Leaflet están embebidos en el HTML.
